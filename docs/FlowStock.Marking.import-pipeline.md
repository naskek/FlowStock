# FlowStock.Marking: конвейер импорта

## Назначение
Документ описывает рабочую русскоязычную схему импорта файлов маркировки.

## Базовый конвейер
1. обнаружить входной файл
2. перенести его в зону обработки
3. вычислить hash файла
4. проверить дублирование импорта
5. определить формат и разделитель
6. распарсить строки
7. нормализовать коды и метаданные
8. выполнить валидацию
9. связать данные с заказом или отправить на ручной разбор
10. сохранить запись об импорте и принятые коды в БД
11. обновить статусы и переместить файл в финальную папку
