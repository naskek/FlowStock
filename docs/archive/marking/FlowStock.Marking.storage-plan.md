# FlowStock.Marking: план хранения

## Назначение
Документ фиксирует рабочий план хранения данных для `FlowStock.Marking`.

## Базовый подход
- новые сущности маркировки добавляются в основную БД FlowStock
- схема развивается через обычные SQL-миграции
- доступ к данным идет через backend/data-access слой, а не через внешний print-runtime
