# Контракт сервера: удаление заказа

## Назначение
Документ фиксирует серверный контракт для сценария 'удаление заказа'.

## Общая схема
- клиент обращается к canonical endpoint-ам сервера
- сервер выполняет валидацию и бизнес-логику
- ошибки возвращаются стабильными кодами, пригодными для UI и тестов

## Что обязательно сохранять
- корректную обработку обязательных идентификаторов
- защиту от невалидных переходов состояния
- совместимость с WPF, TSD и PC web там, где сценарий общий
- журналирование и traceability операции
