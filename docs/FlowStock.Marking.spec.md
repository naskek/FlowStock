# FlowStock.Marking: спецификация

## Назначение
Это русскоязычная рабочая спецификация домена `FlowStock.Marking` в его текущем исследовательском состоянии.

## Область охвата
- ручное и полуавтоматическое хранение данных маркировки
- импорт файлов с кодами
- связь кодов с заказами и печатными batch-ами
- подготовка к дальнейшей backend-реализации
